## 一、新增公共组件

### 1.1 `RecordActionButtons.vue` — 记录操作按钮

**涉及文件**：`src/shared/ui/RecordActionButtons.vue`

**用途**：封装表格记录中的「查看 / 编辑 / 删除」三个操作按钮，根据 `row.status !== 'approved'` 自动隐藏编辑和删除。

**Props / Emits**：

- `row: T`（T 需含 `status: string`）
- `view`, `edit`, `remove`

**使用示例**：

```vue
<RecordActionButtons :row="row" @view="view" @edit="edit" @remove="remove" />
```

---

### 1.2 `DictColumn.vue` — 表格字典列

**涉及文件**：`src/shared/ui/DictColumn.vue`

**用途**：将字典值转换为可读标签，替代每个表格列里重复的 `options.find(t => t.value === row.xxx)?.label || row.xxx`。

**Props**：

- `value: string`
- `options: readonly Option[]`

**使用示例**：

```vue
<DictColumn :value="row.competitionType" :options="COMPETITION_TYPES" />
```

---

### 1.3 `ProofUpload.vue` — 佐证材料上传

**涉及文件**：`src/shared/ui/ProofUpload.vue`

**用途**：统一「佐证材料」上传区域，支持 `v-model:file-list`、自定义提示文案和 accept。

**Props / Emits**：

- `modelValue?: UploadUserFile[]`
- `accept?: string`
- `tip?: string`
- `update:modelValue`

**使用示例**：

```vue
<ProofUpload v-model:file-list="form.proofMaterials" tip="支持 pdf、doc、docx 格式" />
```

## 二、改造 `ApplicationFormRecord.vue`

**涉及文件**：`src/shared/ui/ApplicationFormRecord.vue`

**改动内容**：

1. 操作列改用 `RecordActionButtons` 组件，移除内部写死的操作按钮。
2. 新增 `showAlert?: boolean` prop（默认 `true`），用于科研之星等 Tab 子组件中避免重复 Alert。
3. 修复泛型组件在 `el-table` slot 中的类型推断问题，对 `row` 做 `as T` 断言。
