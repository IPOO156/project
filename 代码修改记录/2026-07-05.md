## 二、新增/修改记录

### 2.1 新增全局主题 Store

**涉及文件**：`src/app/stores/theme.ts`（**新增**）

**内容**：

- 定义 `ThemeMode = 'light' | 'dark'`
- 实现 `readTheme()` / `writeTheme()` / `applyTheme()`，持久化到 `localStorage`，通过 `data-theme` 属性驱动 CSS 变量
- 提供 `setTheme(mode)` 与 `toggleTheme()` 接口
- Store 初始化时立即应用主题，避免刷新闪烁

### 2.2 新增主题样式文件

**涉及文件**：`src/assets/styles/themes.css`（**新增**）

**内容**：

- 定义日间模式默认变量（与 Element Plus 默认主题一致）
- 定义夜间模式覆盖变量（背景、文字、边框、填充色等）
- 针对 Element Plus 组件级颜色 token 进行暗色覆盖

### 2.3 注册主题 Store

**涉及文件**：`src/app/stores/stores.ts`（**修改**）

**修改内容**：新增 `export { useThemeStore } from './theme'`。

### 2.4 注入主题样式

**涉及文件**：`src/main.ts`（**修改**）

**修改内容**：在 `global.scss` 之前导入 `./assets/styles/themes.css`，确保 CSS 变量优先级正确。

### 2.5 HeaderBar 集成主题切换与头像同步

**涉及文件**：`src/app/layouts/components/HeaderBar.vue`（**修改**）

**修改内容**：

- 引入 `useThemeStore`，添加日间/夜间切换按钮（Sun / Moon 图标）
- `el-avatar` 绑定 `userStore.avatar`，实现本地头像与 Header 同步
- 切换按钮使用 `el-tooltip` 提示当前模式

### 2.6 用户 Store 支持头像缓存

**涉及文件**：`src/app/stores/user.ts`（**修改**）

**修改内容**：

- 新增 `AVATAR_CACHE_KEY` 与 `readAvatarCache()` / `writeAvatarCache()`
- 新增 `cachedAvatar` 状态与 `avatar` computed（优先使用后端头像，回退本地缓存）
- 新增 `updateAvatar(avatarUrl)` 方法，同步更新状态与 `localStorage`
- `setUserInfo` 时若后端未返回头像则合并本地缓存
- `logout` 时清理头像缓存

### 2.7 快捷入口逻辑增强

**涉及文件**：`src/features/dashboard/composables/useQuickEntries.ts`（**修改**）

**修改内容**：

- 扩展入口池至 8 个：学科竞赛、社会实践、奖学金、成长时间轴、消息中心、个人档案、荣誉奖项、申报中心
- 首页默认展示 6 个入口
- 新增 `refreshPool()`：从未展示且未隐藏的入口中随机补足
- 新增 `toggleHidden(id)`：显示/隐藏入口，至少保留 1 个可见入口
- 新增 `updateOrder(orderedIds)`：持久化用户拖拽自定义顺序
- 新增 `resetToDefault()`：恢复默认池与顺序
- 保留最近点击历史，用于标识「最近使用」

### 2.8 Dashboard 主页面重构

**涉及文件**：`src/features/dashboard/Dashboard.vue`（**修改**）

**修改内容**：

- 左侧区域重构为多维度画像评估（雷达图 + 指标摘要）

- 快捷入口支持 HTML5 拖拽排序
- 最近动态「查看全部」跳转 `/messages/activities`

## 三、影响范围

- **主题模式**：全站支持 light/dark 切换，偏好持久化到 localStorage，刷新不闪烁；HeaderBar 提供切换入口。
- **快捷入口**：Dashboard 右侧展示 6 个入口，支持刷新随机替换、拖拽排序、显示/隐藏配置；配置保存在 localStorage。
- **多维度画像**：Dashboard 左侧集成雷达图、绩点趋势折线图、奖项气泡图、AI 短板识别、教师建议，数据为 mock，后续对接接口。

- **组件文档**：新增业务组件与 Composable 已登记到对应文档。
