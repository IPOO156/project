## 二、新增/修改记录

### 2.1 新增菜单配置集中管理文件

**涉及文件**：`src/shared/constants/menu.ts`（**新增**）

**内容**：

- 集中定义侧边栏菜单结构、路径与图标
- 导出 `findMenuItemByPath` 函数，供标签栏根据路由匹配图标

### 2.2 标签栏状态支持图标与拖拽排序

**涉及文件**：`src/app/stores/tabs.ts`（**修改**）

**修改内容**：

- `NavTab` 接口新增 `icon?: Component` 字段
- `MAX_TABS` 从 12 调整为 7
- `addTab` 改为返回 `boolean`：已达上限且为新 tab 时返回 `false`
- 新增 `reorderTabs(newOrder)` 方法，支持拖拽后重新排序
- `readStorage` 中增加按 `path` 去重

### 2.3 标签栏 Hook 接入菜单图标与上限警告

**涉及文件**：`src/shared/composables/useTabs.ts`（**修改**）

**修改内容**：

- 新增 `resolveIcon(path)`：根据路由路径从菜单配置匹配图标
- 添加 tab 时传入 `icon`
- 达到上限时通过 `ElMessage.warning` 提示用户先关闭部分标签
- 增加 300ms 防抖，避免警告重复弹出
- 标签满时通过 `router.replace` 回退到上一个激活标签
- `/applications` 使用 `to.path` 作为 tab 键，内部模块切换不再新增标签

### 2.4 Sidebar 使用统一菜单配置

**涉及文件**：`src/app/layouts/components/Sidebar.vue`（**修改**）

**修改内容**：

- 从 `@/shared/constants/menu` 导入 `menuItems`
- 删除本地菜单定义和图标导入，保持侧边栏与标签栏图标来源一致

### 2.5 标签栏组件重构：Edge 风格、图标、拖拽

**涉及文件**：`src/app/layouts/components/NavTabs.vue`（**修改**）

**修改内容**：

- 使用 CSS Flex + CSS 变量 `--tab-base-width` / `--tab-min-width` 实现动态压缩
- 移除左右翻页按钮，改为横向滚动
- 修复关闭按钮默认隐藏但仍拦截点击的问题：`pointer-events: none` / hover 时 `auto`
- `activate` / `close` 中增加 `try/catch` 与 `isNavigationFailure` 错误处理
- 激活 tab 时自动 `scrollIntoView`
- 标签标题左侧显示页面图标；图标缺失时 fallback 到菜单配置
- 实现 HTML5 拖拽排序，拖拽中有半透明和放置目标高亮
- 固定标签（affix）不可拖动，也不允许被放置到其前面

### 2.6 ApplicationHub 模块切换即时响应

**涉及文件**：`src/features/applications/ApplicationHub.vue`（**修改**）

**修改内容**：

- `activeKey` 从 `computed` 改为 `ref` + `watch`，模块按钮点击立即切换本地状态
- 使用 `router.replace` 异步更新 URL，避免阻塞 UI
- 模块切换不再新增"个人档案信息申报"标签

### 2.7 个人中心子菜单补充图标

**涉及文件**：`src/shared/constants/menu.ts`（**修改**）

**修改内容**：

- "个人档案信息"添加 `IdCard` 图标
- "职业规划"添加 `Compass` 图标

## 三、影响范围

- **标签栏交互**：点击响应正常，关闭按钮不拦截 tab 点击；导航失败有日志反馈。
- **标签栏视觉**：多个标签时自动等比压缩，最低 80px；hover 显示完整标题；标签左侧显示对应页面图标。
- **标签数量**：最多保留 7 个标签，超出时阻止打开新页面并弹窗提示。
- **拖拽排序**：普通标签可拖动排序，固定标签始终钉在最左侧。
- **个人档案信息申报**：内部模块切换只保留一个标签，切换更流畅。
- **侧边栏导航**：菜单点击不再重复触发路由，标签上限警告只弹一次。
- **个人中心菜单**：子菜单项现在显示与功能主题匹配的图标。
