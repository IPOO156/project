## 二、新增/修改记录

### 2.1 AI 助手路由补全

### 2.2 暗色模式全局适配——硬编码颜色替换

**修改原因**：多个页面存在 `#1e3a5f`（深海蓝）硬编码，在暗色模式下与深色背景融为一体。

**涉及文件**：

| 文件                                               | 修改                                                                                                                                                                                                      |
| -------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `src/features/profile/ProfileInfo.vue`             | `.card-header__left`、`.profile-user__name`、`.grade-item__num`、`.interest-item__header` 颜色从 `#1e3a5f` 改为 `var(--el-text-color-primary)`；`.card-header__tag` 背景改为 `var(--el-fill-color-light)` |
| `src/features/messages/MessageCenter.vue`          | 图标背景从 `#1e3a5f` 改为 `var(--el-color-primary)`；标题和统计数字改为 `var(--el-text-color-primary)`；未读卡片背景和边框改为 CSS 变量                                                                   |
| `src/features/applications/ApplicationHub.vue`     | 标签激活态边框/文字/徽章背景从 `#1e3a5f` 改为 `var(--el-color-primary)`                                                                                                                                   |
| `src/features/ai-chat/components/AIChatDrawer.vue` | 用户气泡背景从 `#1e3a5f` 改为 `var(--el-color-primary)`                                                                                                                                                   |
| `src/features/awards/AwardOverview.vue`            | 「总提交数」图标颜色按主题动态切换：日间 `#1e3a5f`，夜间 `#60a5fa`                                                                                                                                        |

### 2.3 暗色模式——个人档案页面专项修复

**涉及文件**：`src/features/profile/ProfileInfo.vue`（**修改**）

**修改内容**：

- 多维度画像 `dimensions` 从固定 `ref` 改为 `computed`，根据 `themeStore.isDark` 动态切换 5 个维度的颜色（学业成绩 夜间 `#60a5fa`、竞赛实践 `#34d399` 等）
- 夜间模式下期末成绩卡片显式指定背景、边框、文字颜色
- 多维度画像分数文字夜间强制使用 `var(--el-text-color-primary)`

### 2.4 成长时间轴 UI 修正

**涉及文件**：

- `src/features/growth-timeline/GrowthTimeline.vue`（**修改**）
- `src/features/growth-timeline/components/SummarySection.vue`（**修改**）

**修改内容**：

- `.timeline-sync` 新增 `margin-top: 1.5rem`，解决「每一圈年轮，都有一段故事」与同步控件间距过紧
- 底部四张统计卡片从统一棕色改为四色独立配色（蓝/绿/金/紫），日间和夜间各有对应色值
- 暗色模式选择器从 `[data-theme='dark']` 统一为 `html.dark`，与项目主题系统一致

### 2.5 删除操作统一加确认弹窗

**修改原因**：多处删除操作直接执行，缺少防误触确认。

**涉及文件**：

| 文件                                              | 新增弹窗操作                                                  |
| ------------------------------------------------- | ------------------------------------------------------------- |
| `src/features/profile/ProfileInfo.vue`            | `deleteInterest()`、`deleteAward()` 加 `ElMessageBox.confirm` |
| `src/features/messages/MessageCenter.vue`         | `deleteItem()`、`handleBatchDelete()` 加确认弹窗              |
| `src/features/growth-timeline/GrowthTimeline.vue` | `handleDeleteExperience()` 直接删除路径加确认                 |

### 2.6 首页雷达图垂直居中

**涉及文件**：`src/features/dashboard/Dashboard.vue`（**修改**）

**修改内容**：

- `.radar-panel` 从 `align-items: stretch` 改为 `align-items: center`，雷达图与右侧指标概要在垂直方向居中对齐

### 2.7 表单必填校验统一

**修改原因**：10 个申报表单的模板中标记了 `required`，但提交时未做前端校验，空表单可直接提交。

**涉及文件**：

- `src/shared/composables/useApplicationForm.ts`（**修改**）— 新增 `requiredFields` 选项和 `validateRequired()` 方法，提交前校验必填字段
- 10 个申报表单各传 `requiredFields`：
  - `src/features/applications/competition/CompetitionList.vue`
  - `src/features/applications/innovation/InnovationList.vue`
  - `src/features/applications/research/ResearchList.vue`
  - `src/features/applications/scholarship/ScholarshipList.vue`
  - `src/features/applications/certificate/CertificateList.vue`
  - `src/features/applications/internship/InternshipList.vue`
  - `src/features/applications/organization/OrganizationList.vue`
  - `src/features/applications/training/TrainingList.vue`
  - `src/features/applications/social-practice/SocialPracticeList.vue`
  - `src/features/applications/book-report/BookReportList.vue`
