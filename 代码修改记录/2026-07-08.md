## 二、新增/修改记录

### 2.1 AI 助手页面布局调整

**涉及文件**：`src/features/ai-chat/AIChat.vue`（**修改**）

**修改内容**：

- `.page-container__body`：移除 `min-height` 改为 `height: calc(100vh - #{$header-height} - 50px)`，配合 `box-sizing: border-box` 消除纵向滚动条
- `.ai-chat`：`max-width` 从 1320px 扩大至 1440px
- `.ai-chat__header`：`padding` 从 `28px 32px` 收紧为 `24px 28px`，`margin-bottom` 从 40px 收紧为 28px
- `.ai-chat__card`：`max-height` 从 620px 收紧为 600px
- `.ai-chat__message`：AI 回复气泡 `max-width` 收窄至 `min(640px, 62%)`，用户消息气泡为 `min(760px, 78%)`

### 2.2 成长时间轴卡片右边缘元素对齐

**涉及文件**：`src/features/growth-timeline/components/TimelineNode.vue`（**修改**）

**修改内容**：

- 右下角序号从 `::before` 伪元素改为真实 DOM `<span class="card-index">`
- 新增 `.card-right-edge` 父容器，将右上角圈 + 垃圾桶 + 右下角序号包入统一容器
- 使用 `flex-direction: column; justify-content: space-between; align-items: flex-end` 确保三个元素共享同一右边缘参考线
- 移动端适配同步调整（`.card-right-edge` 的 `top/bottom/right/width` 和 `.card-index` 的 `font-size`）

### 2.3 头像裁剪最小缩放改为 30%

**涉及文件**：`src/features/profile/components/AvatarUploader.vue`（**修改**）

**修改内容**：

- `calculateMinScale` 中的兜底最小值从 `0.5` 降为 `0.3`，允许用户将图片缩小至 30% 后裁剪

### 2.4 成长时间轴同步经历过滤

**涉及文件**：`src/features/growth-timeline/composables/useGrowthDataSources.ts`（**修改**）

**修改内容**：

- 在候选经历合并后新增过滤步骤，排除标题或描述中包含"已报名"或"已记录"的条目（如"组织履历申报已记录""学术研究申报已记录"等状态性描述不清的记录不再同步到时间轴）

### 2.5 温馨提示弹窗触发逻辑修复

**涉及文件**：

- `src/features/dashboard/composables/useGreeting.ts`（**修改**）
- `src/shared/ui/GreetingToast.vue`（**修改**）

**修改内容**：

## 三、影响范围

- **AI 助手页面**：布局更紧凑，回复气泡更窄，整体在可视区域内居中，无额外滚动条。
- **成长时间轴**：卡片右上角圈/垃圾桶与右下角序号视觉上精确对齐；同步其他模块经历时自动过滤不含实质内容的状态性记录。
- **个人档案头像**：裁剪时可缩小至原图的 30%（原 50%），大尺寸图片裁剪更灵活。
- **温馨提示**：登录后或刷新页面时均能正常弹出（每天仅一次），视觉更醒目。
