| `src/app/layouts/components/Sidebar.vue`                 | 自动生效 | 菜单数据来自 moduleRegistry                    |

**核心功能**：

- **申报审核**：Tab 栏含"全部"+10 种独立类型（学科竞赛/创新创业/学术研究/奖学金/荣誉证书/实习经历/组织履历/实训项目/社会实践/图书心得），"全部"Tab 显示通用列+过滤，独立 Tab 按类型动态渲染列和过滤条件。查看弹窗展示完整的类型特有字段，编辑弹窗内转为可编辑表单（dictOptions→el-select、date→el-date-picker、其他→el-input）。
- **奖项审核**：Tab 栏含"全部"+竞赛之星/科研之星/双创之星，查看弹窗按类型展示完整填报信息，编辑弹窗按类型显示对应字段（竞赛之星：竞赛名称/时间/级别/获奖等级/学期；科研项目：项目名称/级别/排名/时间；软著：软件名称/颁发单位/排名/批准日期；论文：论文名称/期刊/排名/发表日期；双创之星：公司名称/行业类型/排名/注册时间）。
- **操作逻辑**：已驳回/待审核→显示"编辑"按钮，可修改后重新提交；已通过→显示"查看"按钮，只读详情。

**补充字段**（review-columns.ts）：

| 类型     | 补充字段                                           |
| -------- | -------------------------------------------------- |
| 创新创业 | `teamRole`（团队角色）                             |
| 学术研究 | `researchType`（研究类型）、`teamRole`（团队角色） |
| 社会实践 | `volunteerHours`（志愿时长）                       |
